# 🛡️ Gök Siper (Frekans Bozan Drone) Projesi

Bu proje,**Tuğrul Murathan Tapar** tarafından geliştirilen, insansız hava araçlarına,yasa dışı haberleşme vb. durumlara karşı savunma amaçlı bir frekans karıştırıcı (jammer) sistemidir.

## 🚀 Proje Amacı
Gök Siper, belirlenen frekans bantlarında sinyal bozucu yayın yaparak izinsiz droneların kontrolünü veya izinsiz haberleşmeleri  kesmeyi amaçlar.

## 🛠️ Teknik Özellikler
- **Modüler Tasarım:** Kolayca drone üzerine monte edilebilir veya sabit istasyon olarak kullanılabilir.
- **Desteklenen Frekanslar:** (300-348 Mhz,387-464 MHz,779-928 MHz,850 MHz,900 MHz,1800 MHz,1900 MHz ve 2.4Ghz )(Frekanslar anten ve yazılım güncellemesiyle artırılabilir.).
- **Yüksek Verimlilik:** Düşük enerji tüketimi ile geniş kapsama alanı.
